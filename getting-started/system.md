## MacOS

- Bash/Zsh

## Linux

- **Ubuntu/Kubantu/Debian/Synology...** : Bash/Zsh
- **Arch/Manjaro/Arco...** : Bash/SH/Zsh
- **RedHat/CentOS...** : Bash/Zsh
- **Deepin...** : Bash/Zsh
- **Other Linux Kernel** : Bash/Zsh

You can run OhMySH in these systems and these shells.
